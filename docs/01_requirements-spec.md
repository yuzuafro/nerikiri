## 1. 背景・目的

### 1.1 背景

和菓子の練り切りは、餡を主原料とした生地を手やヘラで成形し、季節の花や風物を表現する伝統的な上生菓子である。練り切りの成形技術の習得には反復練習が必要だが、実際の生地を使った練習には材料費・準備・後片付けのコストがかかり、気軽に試行錯誤できる環境が少ない。

### 1.2 目的

Webブラウザ上で粘土細工のように3D形状を自由に変形できるシミュレーションアプリケーションを提供する。将来的には練り切りの成形練習・デザイン検討・教育用途に利用できるアプリケーションへ発展させる。

## 2. 対象ユーザー

| ユーザー区分 | 説明 |
|---|---|
| 一般ユーザー | 粘土細工感覚で3D造形を楽しみたい人 |
| 和菓子職人・見習い | 練り切りのデザイン検討・成形イメージの確認をしたい人 |
| 教育関係者 | 和菓子文化・造形の授業教材として利用したい人 |

## 3. ユーザー要求一覧

| 要求ID | 要求内容 | 優先度 |
|---|---|---|
| UR-01 | Webブラウザだけで利用でき、インストールが不要であること | 必須 |
| UR-02 | 画面上に粘土の塊が3Dで表示され、いろいろな角度から眺められること | 必須 |
| UR-03 | マウス操作で粘土を「押す」「引っ張る」「なめらかにする」など直感的に変形できること | 必須 |
| UR-04 | 変形の道具(ブラシ)の大きさや強さを調整できること | 必須 |
| UR-05 | 操作を間違えたときに「元に戻す」「やり直す」ができること | 必須 |
| UR-06 | 練り切りらしい色(白・桃・抹茶など)を粘土全体や一部分に付けられること | 必須 |
| UR-07 | 最初からやり直せる(初期状態に戻せる)こと | 必須 |
| UR-08 | 作った形を保存し、後で読み込んで続きから作業できること | 推奨 |
| UR-09 | 練り切りの成形工程に即した初期形状を選べること: 丸めた直後の「球」と、上面が平らになりつつ下が少しすぼんだ「基本形」 | 推奨 |
| UR-10 | 特別な操作説明がなくても使い始められる、日本語のシンプルな画面であること | 必須 |
| UR-11 | 一般的なPCで滑らかに動作すること | 必須 |
| UR-12 | どこが正面か、粘土の高さがどの程度かが分かる基準(作業台・正面の目印・高さの目盛り等)が画面内に表示されること | 必須 |

| 制約ID | 内容 |
|---|---|
